# SaleMaker

Приложение для заполнения реестра Excel по справочникам, чек‑листам и фотографиям оборудования (PySide6).

## Требования

- Python 3.9–3.13
- Зависимости: `pip install -r requirements.txt`

## Быстрый старт

1. Скопируйте пример конфигурации:
   ```bash
   copy config.example.yaml config.yaml
   copy checklist_templates.example.yaml checklist_templates.yaml
   ```
   (на Linux/macOS: `cp config.example.yaml config.yaml`)

2. Укажите пути к Excel‑файлам и папке фото в `config.yaml`.

3. Запуск:
   ```bash
   python main.py
   ```

## Документация

Подробная инструкция: [Инструкция пользователя.md](Инструкция%20пользователя.md)

## Основные возможности

- Поиск по серийному номеру и номеру п/п в справочнике Excel
- Чек‑листы по типу оборудования с значениями `default` в `config.yaml`
- Шаблоны чек‑листа по модели в отдельном файле `checklist_templates.yaml`
- Справочник цен: производитель/модель, автоподстановка типа оборудования
- Фотографирование с камеры, в том числе полноэкранное превью
- Редактор конфигурации (вкладка в приложении)
- Формирование ZIP‑пакета для отправки
